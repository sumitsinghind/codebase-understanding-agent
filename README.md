# 🧠 AI Codebase Understanding Agent

### LLM-Powered Developer Tool for Intelligent Codebase Exploration

An **AI-powered developer assistant** that analyzes an entire codebase and answers architectural and implementation questions using **Retrieval-Augmented Generation (RAG)**.

This system enables developers to **interact with complex codebases in natural language**, dramatically improving **code understanding, onboarding, and debugging workflows**.

---

## 🚀 Motivation

Understanding a large codebase is one of the hardest challenges for developers. New developers often spend **hours or days exploring unfamiliar repositories** before they understand system architecture, code dependencies, API flows, and business logic.

This project solves that problem by building an **AI agent capable of reasoning about source code**. Instead of reading hundreds of files manually, developers can simply ask context-aware questions and receive explanations generated by an LLM.

> **“Explain the backend architecture.”**

---

## 🧠 Key Idea

The system combines **semantic code search and large language models** to build an AI system capable of understanding software repositories.

It works through the **Retrieval-Augmented Generation (RAG)** architecture by:

1. Converting source code into embeddings
2. Storing them in a vector database
3. Retrieving relevant code fragments
4. Providing context to an LLM
5. Generating human-readable explanations

---

## ⚡ Example Queries

Developers can ask questions like:

- _"Explain the architecture of this backend"_
- _"Where is authentication implemented?"_
- _"What does GameSessionService do?"_
- _"How does the leaderboard system work?"_

The AI retrieves relevant code fragments and generates explanations based on the actual implementation.

## ⚙️ Setup & Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sumitsinghind/codebase-understanding-agent.git
cd codebase-understanding-agent

```

\*\*

### 2️⃣ Create a Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate

```

\*\*

### 3️⃣ Install Dependencies

```bash
pip install langchain langchain-community langchain-text-splitters langchain-huggingface sentence-transformers streamlit faiss-cpu ollama

```

\*\*

### 4️⃣ Install Local LLM (Ollama)

1. Download Ollama from [https://ollama.com/download](https://ollama.com/download)
2. Download the Llama3 model:

```bash
ollama pull llama3

```

\*\* 3. Test the model:

```bash
ollama run llama3

```

\*\*

### 5️⃣ Index the Codebase

Place the source code you want to analyze inside the `repo_files/` directory. Then run:

```bash
python ingest.py

```

\*\*

This step will read all source files, split them into semantic chunks, generate embeddings, and store them in a FAISS vector database.

### 6️⃣ Run the Application

Start the AI interface:

```bash
python -m streamlit run app.py

```

\*\*

Open your browser and navigate to `http://localhost:8501` to start asking questions about the codebase.

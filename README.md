# 🧠 AI Codebase Understanding Agent

### LLM-Powered Developer Tool for Intelligent Codebase Exploration

![Python](https://img.shields.io/badge/Python-3.10+-blue)

An **AI-powered developer assistant** that analyzes an entire codebase and answers architectural and implementation questions using **Retrieval-Augmented Generation (RAG)**.

This system enables developers to **interact with complex codebases in natural language**, dramatically improving **code understanding, onboarding, and debugging workflows**.

---

# 🚀 Motivation

Understanding a large codebase is one of the hardest challenges for developers.

New developers often spend **hours or days exploring unfamiliar repositories** before they understand:

- System architecture
- Code dependencies
- API flows
- Business logic

This project solves that problem by building an **AI agent capable of reasoning about source code**.

Instead of reading hundreds of files manually, developers can simply ask:

> **“Explain the backend architecture.”**

and receive a **context-aware explanation generated by an LLM**.

---

# 🧠 Key Idea

The system combines **semantic code search + large language models** to build an **AI system capable of understanding software repositories**.

It works by:

1. Converting source code into embeddings
2. Storing them in a vector database
3. Retrieving relevant code fragments
4. Providing context to an LLM
5. Generating human-readable explanations

This architecture is known as **Retrieval-Augmented Generation (RAG)**.

# ⚙️ Setup & Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/codebase-understanding-agent.git
cd codebase-understanding-agent
````

### 2️⃣ Create Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3️⃣ Install Dependencies

```bash
pip install langchain
pip install langchain-community
pip install langchain-text-splitters
pip install langchain-huggingface
pip install sentence-transformers
pip install streamlit
pip install faiss-cpu
pip install ollama
```

---

# 🤖 Install Local LLM

Install **Ollama**

```
https://ollama.com/download
```

Download the LLM model:

```bash
ollama pull llama3
```

Test the model:

```bash
ollama run llama3
```

---

# 📥 Index the Codebase

Place the source code you want to analyze inside:

```
repo_files/
```

Then run:

```bash
python ingest.py
```

This step will:

- Read all source files
- Split them into semantic chunks
- Generate embeddings
- Store them in a FAISS vector database

---

# 🖥 Run the Application

Start the AI interface:

```bash
python -m streamlit run app.py
```

Open in your browser:

```
http://localhost:8501
```

Now you can ask questions about the codebase.
